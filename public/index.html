📚 Dokumentasi API (Kirim Pesan)

Gunakan format POST request untuk mengirim pesan lewat aplikasi eksternal (MikroTik/Laravel). Wajib menyertakan Basic Auth di pengaturan request Anda.

Kirim Pesan ke Nomor Personal

Endpoint : POST /api/send
Parameter:
- sessionId : (contoh: kasir-01)
- phone     : (contoh: 08123456789 atau 628123456789)
- message   : (Isi pesan Anda)

Kirim Pesan ke Grup WA

Endpoint : POST /api/send
Parameter:
- sessionId : (contoh: kasir-01)
- phone     : [ID_Grup]@g.us (contoh: 120363123456789@g.us)
- message   : (Isi pesan grup)

* Tips: Untuk mengetahui ID Grup, gunakan fitur "Uji Kirim Pesan" di atas. Pilih "Grup WA" dan ID Grup akan muncul di sebelah nama grup pada dropdown.
